Introduction Over the years, bad actors have used the Network Time Protocol (NTP) as a successful DDOS attack vector. Generally speaking, the cause of these attacks is due to NTP mis-configuration. This post will look at how to build and configure an NTP server and provide insight to help keep your NTP server safe. Assumptions 1. You are not exposing this server to the public internet 2. You are running several NTP servers within your network to keep it highly-available.
